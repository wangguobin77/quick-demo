#### 说明
根据[wanandroid](http://www.wanandroid.com/blog/show/2)网站提供的api开发的快应用

#### 功能
* 首页
  * banner展示
  * 最新文章展示(可收藏)
* 体系
  * 一级分类
  * 二级分类
  * 根据分类展示文章(可收藏)
* 我
  * 登录
  * 注册
  * 查看、添加收藏的文章
  * 查看、编辑、添加收藏网站
* 查看文章内容
* 分享文章
